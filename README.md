<h2 id=title_ch03>
    <a href="notes/03/notes.md">
        第03章 文件I/O
    </a>
</h2>

> 本章开始讨论`UNIX`系统，说明可用的文件`I/O`函数————打开文件、读文件、写文件等。`UNIX`系统中的大多数文件`I/O`只需用到`5`个函数：`open`、`read`、`write`、`lseek`以及`close`。然后说明不同缓冲长度对`read`和`write`函数的影响。
